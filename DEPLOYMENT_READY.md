### Option 1: Vercel (Recommended)
1. Push to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Deploy (automatic)

### Option 2: Self-Host
1. Build: `bun run build`
2. Start: `bun start`
3. Serve on port 3000
